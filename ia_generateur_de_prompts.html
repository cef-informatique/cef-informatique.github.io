Générateur de prompt ChatGPT

● Cet outil est conçu pour créer un prompt ChatGPT efficace en utilisant la méthode QQCCPQO.
● C'est-à-dire : "Qui / Quoi / Comment / Combien / Pourquoi / Quand / Où".
● Cette méthode permet de donner toutes les informations utiles à ChatGPT.
● Selon le sujet de la question, il n'est pas forcément nécessaire de remplir tous les champs.
● Le bouton "Exemples de prompts" vous permet de remplir le générateur à partir d'exemples.
